HBomb Camera

This app needs access to your camera and microphone.

Permission Required

Camera or microphone access was blocked. Follow these steps to re-enable:

Chrome / Edge

  1. Click the camera icon in the address bar (top right of URL bar)
  2. Select Allow for both Camera and Microphone
  3. Reload the page

macOS System Settings

  1. Open System Settings → Privacy & Security → Camera
  2. Enable access for your browser
  3. Repeat for Microphone
  4. Reload the page

Windows Settings

  1. Open Settings → Privacy → Camera
  2. Toggle Allow apps to access your camera to On
  3. Repeat for Microphone
  4. Reload the page
